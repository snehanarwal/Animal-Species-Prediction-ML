 Animal Species Prediction Model
 
 Project Overview:

This project is a Machine Learning based Animal Species Prediction System that classifies animal species using supervised learning techniques.
The model was built using multiple algorithms to compare performance and improve prediction accuracy.

 Features:

Data preprocessing and cleaning

Feature extraction

Model training using:

Random Forest

Support Vector Machine (SVM)

Convolutional Neural Network (CNN)

Accuracy comparison of different models

Visualization of results

Final species prediction output

Tech Stack:

Python

Pandas

NumPy

Scikit-learn

TensorFlow / Keras

OpenCV

Matplotlib

 Workflow:

Data Collection

Data Preprocessing

Model Training

Model Evaluation

Prediction & Visualization

SVM	95.2%

 Key Learnings:

Understanding supervised ML techniques
